## üìÇ Task Overview

### üì± Task 1: Mobile App Signup Flow
*Objective:* Design a signup flow that is short, personalized, and secure.

* *Description:* I created a clean and secure authentication flow for a mobile application. The design focuses on minimizing friction for new users while ensuring all necessary data is collected efficiently.
* *Key Features:*
    * Minimalist input fields to reduce cognitive load.
    * Clear error validation and feedback.
    * "Forgot Password" recovery flow.

### üìß Task 2: Email Template
*Objective:* Design a visually appealing and responsive email template.

* *Description:* A professional "Order Confirmation" email template designed for E-commerce. The layout is responsive, ensuring readability on both desktop and mobile devices.
* *Key Features:*
    * *Visual Hierarchy:* Clear distinction between the order summary, shipping details, and cost.
    * *Responsive Layout:* Adapts seamlessly to different screen sizes.
    * *Call to Action:* Prominent buttons to "View Order" or "Track Shipment."

### üçΩ Task 3: Restaurant Menu UI ("Delicio")
*Objective:* Design a restaurant menu that is categorized and easy to navigate.

* *Description:* A digital menu interface for a fine-dining restaurant named "Delicio". The project focuses on appetizing visuals and intuitive categorization to help users find dishes quickly.
* *Key Features:*
    * *Categorization:* distinct tabs for Appetizers, Main Course, and Desserts.
    * *Visual Appeal:* High-quality food imagery with a dark-mode aesthetic.
    * *User Experience:* Simple "Add to Cart" interactions and clear pricing.
